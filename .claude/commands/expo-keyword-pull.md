# Expo Keyword Pull Workflow

1. The user will provide a command like `/expo-keyword-pull [slug]`, for example `/expo-keyword-pull baguette-walkthrough`.
2. Find the source post at `Expo/Posts/[slug].md`. Read the file using the Read tool.
3. Check if `tags[]` in the frontmatter is already populated with more than one non-placeholder entry.
   - If tags are already substantive (more than just `["placeholder"]` or `[]`): report "Tags already present — confirm to regenerate, or stop." and stop unless the user confirms regeneration.
4. Parse `relatedRecipes[]` from the post's frontmatter. For each recipe id listed:
   - Look up its built file at `site/src/content/recipes/[id].md` — read the frontmatter `keywords` field.
   - Collect all keyword values into a set. These are the inherited keywords.
   - If no recipes are listed, the inherited set is empty.
5. Read the full post body. Generate 8–15 tags covering:
   - Culinary techniques mentioned or discussed
   - Themes (historical, narrative, seasonal, thematic)
   - Cultural or historical references
   - Primary ingredients featured
   - Equipment referenced
   - Flavor profiles or cooking methods
   - Concepts from culinary tradition (e.g. brigade-de-cuisine, mise-en-place)
6. Merge the generated tags with the inherited keywords from step 4:
   - Lowercase-hyphenated format for all terms (e.g. `beurre-blanc`, `confit`, `mise-en-place`)
   - Deduplicate (case-insensitive)
   - Aim for a final list of 10–20 terms total after merging; trim obvious overlap or very generic terms if the list grows past 20
7. **Stop Point:** Present the proposed tag list as a formatted YAML array and wait for the user to confirm before writing anything. Do not write automatically.
8. On confirmation, update the `tags:` field in the post's frontmatter at `Expo/Posts/[slug].md` using the Edit tool. Replace the existing `tags: [...]` line with the new list.
   - Do NOT touch `site/src/content/expo/` — those files are ephemeral, regenerated by `prepare-expo.py`.
9. Report which post was updated, how many tags were written, and how many were inherited vs. generated.
