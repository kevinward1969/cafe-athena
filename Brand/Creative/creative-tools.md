# Café Athena — Creative Tools & Resources

**Last Updated:** 2026-06-17

A reference for all creative tools, AI platforms, and asset resources available for Café Athena production work. Used by agents and collaborators to make tool routing decisions without guessing what's available.

---

## Adobe Suite

| Tool | Account | Best for |
|------|---------|----------|
| **Adobe Creative Cloud** | Pro (full) | Master source — Illustrator for identity/vectors, Photoshop for image retouching and optimization, InDesign for print-ready layout |
| **Adobe Express** | Included with CC Pro | Social graphics, quick-turn branded templates, badge/mark export, carousel frames |
| **Adobe Firefly** | Included with CC Pro | AI image generation within the Adobe ecosystem; generative fill; style-consistent outputs for brand-adjacent imagery |
| **Adobe Stock** | Included with CC Pro | Licensed photography and vector illustration — use for reference or sourcing when original photography isn't available |

## AI Platforms

| Tool | Account | Best for |

## Tool Routing Guide

When producing a Café Athena asset, choose the tool based on what the asset needs to be:

| Asset type | Primary tool | Secondary / fallback |
|------------|-------------|---------------------|
| Recipe hero image | Gemini (Visual Director Gem) | Midjourney for style exploration |
| Social graphic (text + image) | Adobe Express | Export from Illustrator |
| Brand mark / identity work | Illustrator (CC) | — |
| Favicon / icon export | Illustrator → PNG | Adobe Express |
| Free vector illustration | VectorStock | Adobe Stock |
| Free editorial photography | PXHere | Adobe Stock |
| Licensed photography | Adobe Stock | PXHere (CC0) |
| AI image generation (on-brand) | Gemini | Firefly, Midjourney |
| Prose / copy / strategy | Claude | — |
| Research / fact-check | Perplexity | Claude |

---

## Notes

- **Adobe Stock licensing:** Stock images must be licensed before use in published assets. Thumbnails from search results are for evaluation only — license via the Adobe connector before editing or publishing.
- **Midjourney:** Basic account; limited fast hours. Use for concept exploration, not final production. Final assets always go through Illustrator or Photoshop.
- **PXHere / VectorStock free assets:** Always verify license terms on individual assets before use in commercial or published contexts. Both sites have mixed license types.
- **Gemini image generation:** Operates through the Visual Director Gem (`Agents/Gemini-Gems/CAFÉ ATHENA - VISUAL DIRECTOR GEM INSTRUCTIONS.md`). Do not prompt Gemini directly for cookbook imagery without the Gem's aesthetic rules active.
