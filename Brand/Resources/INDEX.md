# Brand Resources — Index

Read this file on startup. When a task matches a trigger below, load the referenced document before proceeding.

| File | What it is | Load when… |

| `dm_2019_key_brand_elements.pdf` | Framework defining the core elements of a brand (identity, voice, visual, positioning) | Starting any brand guidelines work or reviewing brand completeness |
| `Developing Your Personal Brand.docx` | Personal brand development methodology | Working on author identity, bio, or positioning Kevin as the face of Café Athena |
| `brand_assessment_tool.xlsx` | Structured audit of current brand state across key dimensions | Beginning a brand review session or identifying brand gaps |
| `brand_strategy_scorecard.xlsx` | Tracks brand development progress against defined goals | Updating BRAND_STATUS.md or reporting on brand build progress |
| `core_values_worksheet.docx` | Worksheet for defining and articulating brand core values | Writing the Overview section of BRAND_GUIDELINES.md |
| `positioning_statement_worksheet.docx` | Framework for crafting a brand positioning statement | Writing the Brand Identity or Acquisition Voice sections of BRAND_GUIDELINES.md |
| `buyer_persona_template.xlsx` | Structured template for building audience personas | Creating any file in `Brand/Personas/` |
| `brand_perception_survey.docx` | Survey template for measuring how the brand is perceived externally | Once audience exists — checking if brand is landing as intended |
