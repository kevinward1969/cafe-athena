# Marketing Resources — Index

Read this file on startup. When a task matches a trigger below, load the referenced document before proceeding.

| File | What it is | Load when… |

| `word_of_mouth_marketing.pdf` | Strategies for organic referral and community growth | Planning audience growth — especially relevant for a food/recipe brand with no paid budget |
| `customer_engagement_framework.pdf` | Framework for building ongoing audience engagement and loyalty | Planning newsletter, social cadence, or repeat visitor strategy |
